## Task manager using React and context API
This is a simple task manager application built using React and the Context API. It allows users to add, view, and manage their tasks efficiently.
### Features
- Add new tasks
- View a list of tasks
- Mark tasks as completed
- Delete tasks
### Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/VortexWanderer9/taskManagerContextAPI.git
    ```
2. Navigate to the project directory:
    ```bash
    cd taskManagerContextAPI
    ```
3. Install the dependencies:
    ```bash
    npm install
    ```
4. Start the development server:
    ```bash
    npm start
    ```
